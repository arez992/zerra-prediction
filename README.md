# ZERRA Prediction

AI-powered multi-sport prediction platform with premium analytics.

## Sprint 1 included

- Next.js app router foundation
- TypeScript
- Tailwind CSS
- Multi-language routing: English, French, Spanish, Arabic
- Premium dark/gold UI
- Home page
- Predictions page
- VIP page
- Login mock
- Dashboard mock
- API-ready prediction data structure

## Run locally

```bash
npm install
npm run dev
```

Open: `http://localhost:3000`

## Push to GitHub

```bash
git init
git branch -M main
git add .
git commit -m "Sprint 1: Next.js foundation for ZERRA Prediction"
git remote add origin https://github.com/arez992/zerra-prediction.git
git push -u origin main
```
